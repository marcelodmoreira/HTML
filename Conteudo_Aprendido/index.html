Criação de links

Para criação de link utilizamos ⇒ <a>
Importante a utilização da tag < hreflang="linguagem" para informar aos tradutores e navegadores qual a linguagem do site que será aberto.

  1. Links Externos

    2. É sugerido que links externos sejam abertos em abas diferente do seu site
    Tag para abrir em aba diferente ⇒ <a target="_blanck" rel="external"

  2. Links Internos

    3. Quando for navegar em links da própria página utiliamos ⇒ <a target="_self">

Variação das tags <target> e <rel>

  • <target>
    • <taregt = "_blanck">
    Define uma aba nova para abertura do site.
    <target = "_self">
    Define a abertura na mesma aba.
  • <rel>
    • <rel = "external">
    Define que o site é externo a sua página.
    <rel = "prev">
    Determina para o Google sua página anterior
    <rel = "next">
    Determina para o Google sua próxima página.

    Iportante ressaltar que visualmente as definições do <rel> não apresentam diferença. Porém para o algorítimo faz sentido.

Criação de Imagens dinâmicas

Importante a criação de imagem dinâmica para que o tamanho da imagem se adapte a resolução da tela do usuário. Isso ajuda na diminuição da negação do seu site e melhora a qualidade do serviço.
É importnte ressaltar que existe uma hierarquia a ser seguida. Sempre os menores devem ser antecessor dos maiores.
Outro ponto importante é que está sendo usado como padrão os Pixels 1000, 700, 300px.
A tag utilizada para essas imagens ⇒ <picture >

Exemplo:

    <source media="(max-width: 350)" srcset="pasta/imagem menor.png" type="image/png">
    <source media="(max-width: 750)" srcset="pasta/imagem media.png" type="image/png">
    img src="pasta/imagem grande" alt="Imagem Dinâmica"

Criação de Audios

Para criação de Audios iremos utilizar a tag< audio>
A estrutura será:

 
    <audio preload="metadata" controls loop>
    < source src="Local onde o audio está armazenado" type="audio/mpeg">
    </audio>

Entendendo as tags acima:

preload
utilizado para definir o carrgemento do audio:
  • auto para carregar o audio por completo.

    Atenção que essa funcionalidade pode pesar seu site. Pois o tamanho do áudio vai influenciar.

  • metadata carrega somente as informações do arquivo
  • none não carrega nenhuma informaçã
controls
Vai ativar o controlador para que o usuário possa dar o play ou pause no audio
Loop
Para que o audio reiniice após o término.
